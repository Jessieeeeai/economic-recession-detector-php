创建仓库后，GitHub会显示推送命令。**或者直接使用下面的命令**:

### 如果在sandbox环境中操作:
```bash

# 添加远程仓库
git remote add origin https://github.com/Jessieeeeai/economic-recession-detector-php.git

git push -u origin main
```

### 如果在本地环境中操作:
